# funkwerk-base

Funkwerk's base docker image:
- based on ubuntu LTS
- uses tini as a minimal init
- xmllint additionally preinstalled

## Usage

docker run -t funkwerk/base https://github.com/

## Version

0.1.0
 - ubuntu 14.04
 - tini 0.10.0
